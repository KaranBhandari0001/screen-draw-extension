# ✏️ Screen Draw Extension

A lightweight browser extension that allows you to draw, highlight, and explain anything directly on any webpage.

---

## 🚀 Why I Built This

While using screen sharing tools like Google Meet, I often faced a problem — it was difficult to highlight or explain specific parts of a website during live calls.

I needed a simple tool to visually explain things on screen.

So I built this extension.

Now I can easily:

* Highlight important text
* Mark specific sections
* Explain things clearly during meetings

---

## ✨ Features

* ✏️ Draw on any website
* 🔘 Floating draggable button (move anywhere on screen)
* 👆 Double click to activate drawing
* ❌ Double click again to clear and disable
* 📜 Canvas moves with page scroll

  * Draw something
  * Scroll up/down
  * Drawing stays in exact position
* ⚡ Lightweight and fast
* 🌐 Works on most websites

---

## 🎮 How to Use

1. Open any website
2. Use the ✏️ floating button
3. Drag it anywhere on the screen
4. Double click to start drawing
5. Double click again to clear and close

## ⚙️ Installation

1. Clone the repository:
   https://github.com/KaranBhandari0001/screen-draw-extension.git

2. Open your browser:
   edge://extensions

3. Enable Developer Mode

4. Click Load Unpacked

5. Select the extension folder

---

## 📌 Notes

* Does not work on browser internal pages (edge://, chrome://)
* Drawing resets on page reload
